Infrastructure as Code: Technologies Comparison

Year 4

The project I am currently working on is the comparison of different Infrastructure as Code technologies. The technologies I am researching are Terraform, Puppet, Chef, Ansible and Salt Stack. The comparison looks at different functions of IaC like provisioning, management, user experience and complexity. This will showcase each technologies strengths and weaknesses in different use cases. I decided on this idea as when researching into IaC there is a lot of general information without anything specfic which technologies are better used in certain use cases.

MQTT Messaging System

Year 2

This project was a MQTT messaging system. The messages would be saved in a document database. The messages were based any topic which in this case was moview reviews. The messages had fields for the title, director, review, rating and genre. The database also had a web front end which would display all the information in the database, functions were added to modify the database contents by adding and removing.
